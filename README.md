# March Forth

March Forth is a modern derivative of Forth. 

Borrowing from APL, Lisp and ML.

## Features

* Namespaces/Modules 🥳
* Static Type System 🤓
* Multiple Dispatch 😎
* Higher-Order Functions 😇
* Array Processing 🤩
* Parallel Processing 😱
* Lazy Evaluation 🤯

Read the [overview](https://github.com/openbohemians/march/wiki/Overview).

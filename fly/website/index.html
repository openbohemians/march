FORTH YAML

Fly is a object-oriented programming language based on FORTH and YAML. It is an exceptionally powerful blend of form and function.

PLEASE NOTE! Fly is ALPHA status. The specification and this website are under construction and changing rapidly.


YAML in Form

YAML ain't markup language. Indeed, YAML is in fact a data serialization syntax. And as anyone knows from Lisp, code is data. Which is exactly what FLy explots by organizing code into YAML data structures, specifically mappings. Fly uses YAML to define prototype object structures.

Only one variation to the current YAML 1.1 standard is neccessary to support Fly. FLy requires string interpolation, both internal and from external files. It is more suitably defined in the YAML specification then as part of Fly's underlying Forth implementation. We hope to get iterpolation into the upcoming 1.2 or 1.3 specs.


FORTH in Function

Fly uses a variant of Forth. It differs in some important ways:

  • Few non-alphanumeric words
  • Uniquely object-oriented
  • Supports collection and string types

Fly's Forth variation, like Forth itself, uses postfix notation. Most people are not used to this syntax having been taught infix and prefix notations in school. Yet postfix notation has some clear advantages once the initial accustomization is made. It has a naturally object-oriented in that objects must be given prior to an operation. Indeed, Fly is in effect "polyobject-oriented". And it naturally applies to stack-based computation which is highly effcient and suitable equally to low-level programming as it is high-level.


Hello Word Example

Getting started with Fly will be increadibly easy. In fact, if you are currently familiar with YAML and Forth, it's should be almost trivial. The main learning curve comes from familiarizing oneself with the variation of words Fly uses and the libraries available.

Here's the traditional example of Hello World written in Fly.

    # The traditional Hello World progam

    --- !fly-program

    main: !fn
      "Hello, World!" console.print


    > Hello, World!
